각 단계 상세

① 문제 생성 기존 generateScore() 그대로 사용. 정답 ScoreNote[]를 state에 저장하되 화면에는 숨김
② 음악 재생 정답 악보를 ABC로 변환 → 기존 AbcjsRenderer로 재생 (무제한). 빈 오선지 + 첫 음 힌트만 표시
③ 악보 입력 음길이 선택 → 건반 탭 → userNotes[] 배열에 추가 → ABC 변환 → WebView에 실시간 렌더링
④ 제출 & 채점 정답 ScoreNote[]와 사용자 ScoreNote[]를 음표별 비교. 음높이 + 음길이 모두 맞아야 정답
⑤ 결과 확인 정답 악보와 내 악보를 나란히 표시. 틀린 음표는 빨강, 맞은 음표는 초록. 정확도 % 표시
⑥ 다음 문제 정확도 기반 selfRating 자동 계산 → 기존 스킬 프로필 업데이트 로직 재활용
