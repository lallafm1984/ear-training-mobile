건반 스펙

전체 범위 C3 ~ C6 (3옥타브, 흰건 22개 + 검은건 15개)
화면 표시 약 10~12건이 화면에 보임, 나머지는 스크롤
초기 위치 선율: C4 중심 / 2성부 베이스: C3 중심 (성부 전환 시 자동 이동)
검은 건반 흰건 사이 위쪽에 짧은 검은 건반 배치 (실제 피아노 비율)
탭 동작 건반 탭 → 선택된 음길이로 ScoreNote 생성 → userNotes[]에 push → ABC 변환 → 악보 업데이트
탭 피드백 건반 누를 때 색상 변화(하이라이트) + 해당 음 짧게 재생 (MIDI preview)

입력 동작 규칙

기본 음표 입력 음길이 선택 상태에서 건반 탭 → ScoreNote 추가
점음표 "점" 버튼 토글 ON → 다음 건반 탭 시 점음표로 입력 → 자동 OFF
임시표 (버튼) ♯/♭ 토글 ON → 흰건반 탭 시 반음 올림/내림 적용 → 자동 OFF
임시표 (검은건) 검은 건반 직접 탭 → ♯/♭은 조표 컨텍스트에 따라 자동 결정
쉼표 "쉼표" 버튼 탭 → 선택된 음길이의 쉼표가 즉시 추가 (건반 탭 불필요)
붙임줄 "붙임줄" 토글 ON → 다음 건반 탭 시 이전 음표와 tie 연결 (같은 음높이여야 함)
Undo 마지막 입력 음표 제거
음표 탭 삭제 악보에서 음표 탭 → 선택 상태 → 삭제/교체 옵션
마디 초과 방지 현재 마디의 남은 박자보다 큰 음길이 선택 시 버튼 비활성화
